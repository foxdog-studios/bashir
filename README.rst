Bashir
======

Execute Bash scripts from the directory they're stored in.

Installation
------------

To install, run

.. code-block:: bash

    $ ./scripts/install.sh

Usage
-----

Replace the standard shabang

.. code-block:: bash

    #!/bin/bash

with

.. code-block:: bash

    #!/usr/bin/bashir
